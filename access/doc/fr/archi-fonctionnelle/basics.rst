Généralités
***********
Le module d'accès permet de réaliser les opérations suivantes :
-Rechercher des unités d'archives via des mots-clés.
-Afficher les résultats d'une recherche.
-Faire une consultation détaillée d'une unité d'archive.
-Modifier les métadonnées d'une unité d'archive
