Storage-offer
#############

Type :
	Composant VITAM Java

Données gérées :
	* Données d'archives

Typologie de consommation de resources :
	* CPU : moyen (principalement dû aux I/O)
	* Mémoire : fort (principelement pour le cache I/O)
	* Réseau : fort (entrant)
	* Disque : fort (stockage pérenne des données d'archive)
